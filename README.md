# SentrycCodingChallenge

Steps to Run the Tests:

Open the git url https://github.com/naqvi-haider/SentrycCodingChallenge.git
Make a separate folder or on terminal use mkdir command to create a folder in your local repository
use command git clone and use the url "https://github.com/naqvi-haider/SentrycCodingChallenge.git" to clone the project in your local repository
After succesfully cloning the project use this command ./node_modules/.bin/cypress open to run the tests.
